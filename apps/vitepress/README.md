# VitePress

由 Vite 和 Vue 驱动的静态站点生成器

将 Markdown 变成优雅的文档，只需几分钟

- [官网地址](xxx/)



## 基础配置

**安装依赖**

```
pnpm add vitepress@1.6.4

```

```
pnpm vitepress init
```
